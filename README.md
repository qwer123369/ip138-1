## ip138
本人学习python时，使用ip138查询网进行批量查询的练习

获取token: https://user.ip138.com


## 使用方法
1. 在ips.txt中放入需要查询的ip

2. 在codes.txt中放入需要查询的行政区划代码

3. 运行脚本后按提示填入token

## 结果
在同一目录下生成result_ip.txt和result_weather.txt
